# Tennis Match Tracker (Love Game)

A Flutter application for tracking live tennis matches with a sleek, modern UI. This app supports iOS, Android, and Web platforms.

## Features

- View live tennis match scores and statistics
- Browse upcoming and past matches
- Calendar day selection to navigate between dates
- Live score and serving indicators
- Watch live matches with a single tap

## Color Scheme

- Primary Color (Button): #94E831 (Green)
- Background: #000000 (Black)
- Text/UI Elements: #FFFFFF (White)
- Accent: #AC49FF (Purple)

## Project Structure

- `lib/components`: Reusable UI components
- `lib/pages`: Application pages
- `lib/utils`: Utility functions
- `lib/services`: API services
- `lib/assets`: Static resources (images, fonts)
- `lib/locales`: Internationalization files
- `lib/tests`: Test files

## Dependencies

- Flutter
- Provider (State Management)
- Dio (Network Requests)
- flutter_localizations (Internationalization)
- intl (Internationalization)

## Getting Started

1. Make sure Flutter is installed on your machine
2. Clone the repository
3. Run `flutter pub get` to install dependencies
4. Run `flutter run` to start the app on your selected device

## Internationalization

The app supports multiple languages through Flutter's internationalization library. Currently supported languages:

- English
- Spanish
